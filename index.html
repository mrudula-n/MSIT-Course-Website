Introductory Computer Science Course

Course Description

An introductory computer science course for students with little to no computing experience. Students will learn the core elements of programming in Python and will also receive a broad overview of big ideas in the field of computer science...

Learning Outcomes

  • Read and write programs using basic Python syntax.
  • Understand how computers represent concepts through abstraction.
  • Identify appropriate data structures for representing different kinds of data.
